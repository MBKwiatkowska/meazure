One of the primary roles of Meazure is to allow you to easily measure points and objects on the screen. Below are some common measurement tasks.
What do you want to do?
  • Select a measurement tool
  • Measure the position of a point on the screen
  • Measure the distance between two points
  • Measure the size of an image on the screen
  • Measure the size of a window
  • Measure an arbitrary angle
  • Move a crosshair one pixel
  • Determine the accurate screen resolution
  • Determine the size of the screen
  • Determine the color of a pixel
  • Changing coordinate systems
  • Save the position of a measurement tool
