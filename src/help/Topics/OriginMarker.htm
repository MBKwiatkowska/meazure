The Meazure program marks the current location of the origin of the coordinate system using the marker shown on the left. The origin is located at the point where the vertical and horizontal lines meet. The horizontal line designates the x-axis with the positive x direction extending to the right from the origin. The vertical line designates the y-axis with the positive direction extending downward from the origin. Note that if the y-axis is inverted, the origin marker changes its orientation so that the positive y-axis extends upward from the origin as shown in the second figure on the left. If the origin is moved, the marker moves to the new origin. The origin marker can be hidden using the Tools Preferences.
