Meazure can be customized in a variety of ways.
What do you want to do?
  • Hide a section of the program
  • Hide the popup data windows
  • Change the colors of the crosshairs
  • Change the opacity of the crosshairs and popup data windows
  • Change the color of measurement tool lines
  • Change the position of the rulers
  • Change the colors of the rulers
  • Change the opacity of the rulers
  • Use a different screen resolution
  • Change the number of decimal places displayed
  • Create a custom set of measurement units
