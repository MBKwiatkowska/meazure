Detailed Description

An application class mixin inherits from the IValidationHandler interface and overrides the HandleValidationError method.

This method is passed a ValidationError instance whenever the validator detects an invalid construct. The implementor of this interface passes its 'this' pointer to the Validator class when it is constructed.
