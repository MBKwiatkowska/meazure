CString MeaMakeTimeStamp ( time_t  t)

Creates an ISO 8601 compliant time stamp based on the specified time value.

The format of the timestamp is:

yyyy-mm-ddThh:mm:ssZ

Parameters:
t[in] Number of seconds since the Epoch as defined by the system time() function.
Returns:
String containing the timestamp in 8601 format corresponding to the specified time.
