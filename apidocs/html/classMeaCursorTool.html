Detailed Description

Cursor position measurement tool.

This tool measures the position of the mouse pointer.

Constructor & Destructor Documentation

MeaCursorTool::MeaCursorTool ( MeaToolMgr *  mgr) [explicit]

Constructs a new instance of a cursor position measurement tool.

To use a newly constructed Cursor tool, the Enable() method must be called. Enable() will call the Create() method the first time it is called if Create() has not previously been called.

Parameters:
mgr[in] Tool manager parent of the Cursor tool.
