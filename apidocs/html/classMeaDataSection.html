Detailed Description

The data display is divided into two major sections: Region and Screen.

The Region section displays the measurement data from the radio measurement tools (e.g. Point tool, Line Tool). The Screen section displays data about the current monitor (e.g. resolution, width, height).

This class is used to represent each section.
