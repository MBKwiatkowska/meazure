Member Enumeration Documentation

enum MeaPreferences::PreferencePage

Preference page selection indices.

With the exception of kAllPages, these indices must be consecutive and in the order in which pages are to be listed on the property sheet.

Enumerator:
kAllPages 

Refers to all property pages.

kCalibrationPage 

Selects the preference page for screen resolution calibration.

kRulerPage 

Selects the preference page for ruler appearance.

kToolsPage 

Selects the preference page for crosshair and tool appearance.

kPrecisionPage 

Selects the preference page for numerical display precision.

kCustomPage 

Selects the preference page for defining custom measurement units.

kAdvancedPage 

Selects the preference page for setting a startup profile.

BOOL MeaPreferences::OnCommand ( WPARAM  wParam,
LPARAM  lParam 
) [protected, virtual]

Works around a design flaw in the CPropertySheet class.

The current property page is not remembered between invocations of the dialog. Overriding this method allows catpuring the current page so it can be restored the next time the dialog is displayed.

Parameters:
wParam[in] See the CWnd::OnCommand documentation
lParam[in] See the CWnd::OnCommand documentation
Returns:
TRUE if the message has been processed.
