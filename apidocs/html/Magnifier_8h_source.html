00034 /// Provides a screen magnifier window complete with freeze frame, optional
00035 /// pixel grid and pixel color capture. A number of color spaces are available
00036 /// for the display of the captured pixel color. The contents of the screen are
00037 /// read and displayed in this window using a 1X thru 32X zoom factor.
00038 /// Note that layered windows will not appear in the magnifier. Similarly,
00039 /// windows rendered using DirectX will appear black since they are drawing
00040 /// directly to the hardware and displaying using a hardware overlay.
