Detailed Description

Circle measurement tool.

This tool provides a circle and its radius. Two crosshairs are provided: one at the center of the circle (vertex) and one where the radius intersects the perimeter (point 1). Each crosshair can be arbitrarily positioned, which will result in moving and resizing the circle.

void MeaCircleTool::Strobe ( ) [virtual]

Visually strobes the tool.

The tool crosshairs are cycled between a hilite color and their normal color once. In addition, the data windows attached to the crosshairs are cycled between hidden and visible. Strobing the tool is done to visually confirm region capture and position recording.
