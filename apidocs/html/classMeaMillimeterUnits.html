FSIZE MeaMillimeterUnits::FromPixels ( const FSIZE &  res) const [protected, virtual]

Returns the X and Y factors to convert from pixels to millimeters.

In other words, multiplying the values returned from this method by pixels results in a value expressed in millimeters.
