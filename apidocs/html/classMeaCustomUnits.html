enum MeaCustomUnits::ScaleBasis

User defined units are based on a conversion factor.

This factor converts from a basis set of units to the custom units. The basis for the conversion is identified by this enumeration.

Enumerator:
PixelBasis 

Custom units = factor * pixels.

InchBasis 

Custom units = factor * inches.

CentimeterBasis 

Custom units = factor * centimeters.
