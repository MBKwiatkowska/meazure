Detailed Description

A popup data display window.

A data display window is attached to each radio tool's crosshair and a corner of the Window Tool. When the user hovers over a crosshair and while the user drags the crosshair, the data display window is shown and provides a continues readout of the measurements appropriate to the tool. The user can set a preference to not display the data windows.
